PASSWORD RESET PROCEDURES

How to Reset Your Password:

Self-Service Password Reset:
1. Go to the company portal login page
2. Click on "Forgot Password" link
3. Enter your company email address
4. Check your email for reset instructions (check spam folder if not received)
5. Click the reset link in the email (valid for 1 hour)
6. Follow the link to create a new password
7. Enter new password twice to confirm
8. Log in with new password

Password Reset via Phone:
1. Call IT Help Desk at extension 1234
2. Provide your employee ID and manager's name
3. Answer security questions for verification
4. IT will send temporary password via email
5. Log in and change password immediately
6. Temporary passwords expire after 24 hours

Password Reset via IT Portal:
1. Log into IT Support Portal (if you have access)
2. Navigate to "Password Reset" section
3. Select your account
4. Verify identity with security questions
5. Receive reset link via email
6. Follow email instructions to reset

Password Requirements:
- Minimum 12 characters
- Must include uppercase and lowercase letters
- Must include at least one number
- Must include at least one special character (!@#$%^&*)
- Cannot reuse your last 5 passwords
- Cannot contain your username or email
- Cannot contain common words or patterns
- Must be changed every 90 days
- Password complexity meter shows strength

Password Best Practices:
- Use a unique password (not used elsewhere)
- Consider using a passphrase (multiple words)
- Use password manager (company-approved)
- Never share your password with anyone
- Never write passwords down in visible locations
- Change password immediately if compromised
- Use different passwords for different systems

MULTI-FACTOR AUTHENTICATION (MFA)

MFA Setup:
- MFA is required for all company accounts
- Setup during initial account creation
- Can be set up via IT portal or with IT assistance
- Multiple methods available: Authenticator app, SMS, Email

MFA Methods:
- Authenticator App (Recommended): Microsoft Authenticator, Google Authenticator
- SMS: Text message codes to mobile phone
- Email: Codes sent to company email
- Backup codes: Provided during setup, store securely

MFA Best Practices:

- Keep backup codes in secure location
- Register multiple devices if possible
- Update phone number if changed

MFA Troubleshooting:
- Not receiving codes: Check phone/email, try backup method
- Lost device: Contact IT immediately to reset MFA
- Backup codes: Use if primary method unavailable
- New device: Add via IT portal or contact IT

WIFI CONNECTION TROUBLESHOOTING

Company WiFi Networks:
- Company-WiFi: Main network for employees (WPA2-Enterprise)
- Company-Guest: Guest network (password provided at reception)
- Company-IoT: For IoT devices (IT approval required)

Connecting to Company-WiFi:
1. Select "Company-WiFi" network
2. Enter company username and password
3. Accept certificate if prompted
4. Connected successfully

Common WiFi Issues and Solutions:

1. Cannot connect to WiFi:
   - Forget the network and reconnect
   - Restart your device
   - Check if WiFi is enabled
   - Verify you're using the correct network name (Company-WiFi)
   - Check if device is in range of access point
   - Update device WiFi drivers
   - Contact IT if problem persists

2. Slow connection:
   - Move closer to the access point
   - Disconnect and reconnect
   - Check for interference from other devices
   - Limit bandwidth-intensive applications
   - Check if too many devices connected
   - Restart router/access point (if you have access)
   - Contact IT if problem persists

3. Frequent disconnections:
   - Update your device's WiFi drivers
   - Check for software updates
   - Forget network and reconnect
   - Check power management settings (may disconnect to save power)
   - Contact IT Support for advanced troubleshooting
   - May need to update device firmware

4. Authentication errors:
   - Verify username and password are correct
   - Check if account is active
   - Clear saved credentials and re-enter
   - Contact IT if account may be locked

5. Limited connectivity:
   - Restart network adapter
   - Release and renew IP address
   - Check firewall settings
   - Contact IT for network configuration

WiFi Security:
- Always use company WiFi (not public WiFi)
- VPN required when working remotely
- Don't share WiFi password
- Report suspicious networks to IT
- Keep devices updated with latest security patches

TROUBLESHOOTING GUIDE

Common Issues and Solutions:

Email Not Working:
- Check internet connection
- Verify email credentials
- Clear email cache
- Restart email client
- Check if account is locked
- Contact IT if problem persists

Printer Not Working:
- Check if printer is powered on
- Verify network connection
- Check printer queue for errors
- Restart print spooler
- Reinstall printer driver
- Contact IT for assistance

Slow Computer:
- Restart computer
- Close unnecessary applications
- Check available disk space
- Run disk cleanup
- Update software
- Contact IT for performance review

Cannot Access Shared Drive:
- Verify network connection
- Check if VPN is connected (if remote)
- Verify access permissions
- Map network drive again
- Contact IT to verify access

Software Crashes:
- Update software to latest version
- Restart application
- Restart computer
- Clear application cache
- Reinstall software (contact IT)
- Check for known issues in knowledge base
